# Portfolio Wise UI/UX ドキュメント

## 1. プロダクト概要

### ビジョン
Portfolio Wiseは、日本の個人投資家向けに設計された、AIと連携する次世代のポートフォリオ管理アプリケーションです。複雑な投資の世界を、直感的で親しみやすいインターフェースで誰もが使いやすくすることを目指しています。

### ターゲットユーザー
- **プライマリ**: 日本在住の個人投資家（20-60代）
- **投資経験**: 初心者から中級者
- **デバイス**: モバイルファースト（iOS最適化）、デスクトップ対応
- **言語**: 日本語/英語バイリンガル対応

## 2. デザイン原則

### 2.1 ビジュアルデザイン
- **ダークモード中心**: 目に優しく、長時間の使用でも疲れにくい
- **カラーパレット**:
  - Primary: `#3B82F6` (信頼感のあるブルー)
  - Success: `#10B981` (成長を表すグリーン)
  - Danger: `#EF4444` (注意を引くレッド)
  - Background: `#0F172A` (深い紺色のダーク背景)

### 2.2 インタラクション原則
- **即座のフィードバック**: すべてのアクションに視覚的レスポンス
- **スムーズなトランジション**: 200ms標準のアニメーション
- **プログレッシブディスクロージャー**: 必要な情報を段階的に表示

### 2.3 アクセシビリティ
- **高コントラスト**: WCAG AA準拠
- **大きなタップターゲット**: 最小44x44px
- **明確なフォーカス表示**: キーボードナビゲーション対応

## 3. ユーザージャーニー

### 3.1 初回利用者のオンボーディング

```
開始 → Googleログイン → 初期設定ウィザード → AIアドバイザー → ポートフォリオ管理
```

#### 初期設定ウィザード（AIアドバイザー）
1. **基本情報入力**
   - 年齢（スライダーで直感的に）
   - 職業・家族構成（選択式＋カスタム入力）
   - 実現したい夢（プリセット＋自由記述）

2. **投資対象市場選択**
   - カード型UI（視覚的に選択）
   - 人気の組み合わせ提案
   - 複数選択可能

3. **ポートフォリオ・目標設定** 統合ステップ
   - 価値観・不安の選択
   - AIプロンプト生成でポートフォリオデータ収集
   - 現在の資産・理想のポートフォリオ入力

4. **スクリーンショット分析**
   - 証券会社のスクリーンショットをAIで分析
   - プライバシー保護（アップロード不要）

5. **AIプロンプト生成**
   - パーソナライズされた投資アドバイス用プロンプト
   - 外部AI（Claude/Gemini）との連携

### 3.2 リピートユーザーのフロー

```
ログイン → ダッシュボード → 日常的な管理タスク
                     ↓
              定期的な見直し → AI相談
```

## 4. 主要画面とコンポーネント

### 4.1 ダッシュボード
- **ポートフォリオサマリーカード**: 総資産額、損益、分散状況を一目で
- **インタラクティブチャート**: ドーナツチャート、時系列グラフ
- **クイックアクション**: よく使う機能への素早いアクセス

### 4.2 AIアドバイザー
- **ステップ型フォーム**: プログレスバーで進捗を可視化
- **コンテキスト保持**: 入力内容の自動保存
- **リアルタイムプロンプト生成**: 入力に応じて動的に生成

### 4.3 データインポート
- **複数の取り込み方法**:
  - AI分析結果の貼り付け
  - JSONファイルのインポート
  - Google Drive連携
- **リアルタイム検証**: エラーを即座に表示
- **インポート履歴**: 過去の取り込みを追跡

### 4.4 設定画面
- **現在の保有資産エディター**: ドラッグ&ドロップで順序変更
- **目標配分エディター**: ビジュアルなスライダー
- **AIプロンプト設定**: カスタマイズ可能なパラメータ

## 5. インタラクションパターン

### 5.1 フォーム入力
- **プログレッシブエンハンスメント**: 基本→詳細へ段階的に
- **インラインバリデーション**: 入力中にエラーチェック
- **スマートデフォルト**: 一般的な値を事前設定

### 5.2 データビジュアライゼーション
- **インタラクティブチャート**: ホバーで詳細表示
- **レスポンシブデザイン**: デバイスサイズに最適化
- **アニメーション**: データ更新時のスムーズな遷移

### 5.3 エラーハンドリング
- **親しみやすいエラーメッセージ**: 技術用語を避ける
- **回復可能なアクション**: 「やり直す」オプション
- **コンテキスト保持**: エラー後も入力内容を保持

## 6. モバイルファースト設計

### 6.1 タッチ最適化
- **大きなタップエリア**: 指での操作を考慮
- **スワイプジェスチャー**: 自然な操作感
- **下部ナビゲーション**: 親指での操作を優先

### 6.2 レスポンシブレイアウト
- **ブレークポイント**:

- **フレキシブルグリッド**: コンテンツの再配置
- **条件付き表示**: デバイスに応じて要素を表示/非表示

## 7. パフォーマンス最適化

### 7.1 読み込み速度

## 12. デザインシステム

### 12.1 コンポーネントライブラリ
- **ModernButton**: 一貫したボタンスタイル
- **ModernCard**: 統一されたカードデザイン
- **ModernForm**: フォーム要素の標準化
- **React Icons**: Font Awesome、Heroicons等の統一されたアイコンセット

### 12.2 アイコン使用ガイドライン

#### 推奨アイコンライブラリ
```jsx
// Font Awesome (fa)
import { FaRobot, FaChartBar, FaWallet } from 'react-icons/fa';

// Heroicons (hi)
import { HiSparkles, HiChartBar, HiGlobeAlt } from 'react-icons/hi';

// Material Design (md)
import { MdDashboard, MdSettings } from 'react-icons/md';
```

#### アイコン使用原則
- **意味の明確性**: アイコンの意味が直感的に理解できること
- **一貫性**: 同じ概念には同じアイコンを使用
- **サイズ統一**: 基本的に16px、20px、24pxのサイズを使用
- **カラー統一**: テーマカラーに合わせた色使い
- **アクセシビリティ**: 必ずaria-labelやタイトルを併用

#### 禁止事項
- 絵文字の使用（一貫性とアクセシビリティの観点から）
- 複数のアイコンライブラリの混在（統一感の欠如）
- アイコンのみでの情報伝達（テキストラベル必須）

## まとめ

Portfolio Wiseは、複雑な投資管理を誰もが使いやすいものにするという使命のもと、以下を実現しています：

1. **直感的な操作性**: 投資初心者でも迷わない設計
2. **AI との賢い連携**: 外部AIの力を借りて高度な分析を実現
3. **プライバシーファースト**: ユーザーデータの保護を最優先
4. **継続的な改善**: ユーザーフィードバックに基づく進化

このドキュメントは、開発チームとデザイナーが一貫した体験を提供するためのガイドラインとして機能します。
