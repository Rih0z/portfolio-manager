## 現状分析

### 既存 DividendForecast（src/components/dashboard/DividendForecast.tsx）

**構造:**
- `convertToBaseCurrency()` — 通貨換算ヘルパー（9-BX ルール準拠）
- `FREQUENCY_MAP` — 配当支払い月マッピング（monthly/quarterly/semi-annual/annual）
- `MONTH_LABELS` — 月名ラベル（「1月」〜「12月」）
- `AssetDividend` — 銘柄別配当情報 interface
- `MonthlyDividend` — 月別配当 interface
- `DividendForecast` — メインコンポーネント

**機能:**
1. 年間配当合計（Badge表示）
2. 加重平均配当利回り
3. 月別配当バーチャート（12列）
4. トップ配当銘柄（上位5件）

**不足している機能（10-A で追加）:**
1. 月別配当詳細テーブル（どの月にどの銘柄からいくら）
2. 配当利回りランキング（利回り%順）
3. 配当推移グラフ（将来的に過去12ヶ月データが必要 → 現状は予測ベースのみ）

## 変更ファイル

### 1. `src/components/dashboard/DividendForecast.tsx`（MODIFY — 主要変更）

**追加する型:**
```typescript
interface MonthlyDetail {
  month: number;
  label: string;
  assets: {
    ticker: string;
    name: string;
    amount: number;
    frequency: string;
  }[];
  total: number;
}
```

**追加するデータ計算（useMemo 内）:**
- `monthlyDetails: MonthlyDetail[]` — 月ごとに「どの銘柄からいくら」を集計
- `yieldRanking: AssetDividend[]` — 利回り%の降順ソート（全配当銘柄）

**UI 変更:**
- shadcn/ui `Tabs` コンポーネントを追加
- タブ1「サマリー」: 既存のバーチャート + トップ銘柄（変更なし）
- タブ2「月別詳細」: MonthlyDetail テーブル表示
- タブ3「利回りランキング」: 利回り%降順の全銘柄リスト

**通貨換算ルール（9-BX 教訓）:**
- 月別詳細の金額は全て `convertToBaseCurrency` 経由で表示
- ランキングの年間配当額も `baseCurrency` に換算済み

### 2. `src/pages/Dashboard.tsx`（変更なし）

DividendForecast は既にダッシュボードに配置済み。import 変更不要。

## リスク・注意点

1. **バンドルサイズ**: Tabs は shadcn/ui の既存コンポーネントなので追加コスト最小
2. **通貨換算**: 既存の `convertToBaseCurrency` をそのまま使用（新規関数不要）
3. **データ不足**: 過去配当実績データはバックエンドにないため「予測」ベースのまま（将来 API 追加時に切替可能な設計）
4. **配当頻度の精度**: `dividendFrequency` が未設定の場合のデフォルト `quarterly` は既存ロジック踏襲
